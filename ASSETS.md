# Asset licenses

## Quaternius Toon Shooter Game Kit

The following files are derived from the
[Quaternius Toon Shooter Game Kit](https://quaternius.com/packs/toonshootergamekit.html),
which Quaternius released under [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/):

- `assets/models/soldier_ct.glb`
- `assets/models/soldier_t.glb`
- `assets/models/viewmodels/npc-arms-ct.glb` (a body-stripped derivative of
  `soldier_ct.glb`)

The files were processed and exported to GLB for Tiny Strike. Attribution is
not required by CC0, but is included with thanks to Quaternius.

## Tiny Strike original assets

The weapon and equipment GLBs in `assets/models/viewmodels/`, excluding the
Quaternius-derived arm file listed above, were authored for Tiny Strike from
the game's original procedural weapon designs. They are released under the
project's [ISC License](LICENSE).

Textures and game audio are generated from project source code. The trailer
soundtrack is generated by `tools/trailer-audio.js` and is also covered by the
project's ISC License.
